Description: This code is a ROS node that subscribes to the "/odom" topic, which provides odometry information. When a new message is received, the callback function "Cbkodom" is triggered. Inside the callback function, the received position and velocity values are assigned to a custom message of type "robot_r::Vel". The assigned values are then printed to the console using "ROS_INFO". The code also publishes the custom message to the "/pod_vel" topic at a rate of 1 Hz. The loop continues as long as the ROS node is running.
