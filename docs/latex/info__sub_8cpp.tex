Description\+: This code is a R\+OS node that subscribes to two topics\+: \char`\"{}/reaching\+\_\+goal/goal\char`\"{} and \char`\"{}/pod\+\_\+vel\char`\"{}. It extracts the target position from messages received on the \char`\"{}/reaching\+\_\+goal/goal\char`\"{} topic and calculates the average velocity and remaining distance based on position and velocity information received on the \char`\"{}/pod\+\_\+vel\char`\"{} topic. The node continuously updates and prints these values as new messages are received.
